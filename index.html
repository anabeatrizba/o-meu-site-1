Hoje aprendi sobre o que é a triforce da web: HTML, CSS e JavaScript., sendo o foco da aula o HTML.

De uma forma simples, HTML significa HyperText Markup Language e é o que define a estrutura da página, Cascading Style Sheets ou CSS é usada para criar o visual da página, Já o JavaScript é uma linguagem de progamação que progama a interação com a página. Estes três são os principais utilizados na criação de uma página Web.

HTML é um conjunto de "etiquetas" (TAGS) de formatação e existem várias delas. Uma tag inicia-se pelo uso de < > e, quando acaba, finaliza-se com <\/ > . Uma tag pode conter outras tags dentro.

Heading é utilizada para indicar um título e é definido por h1 até h6. Quanto menor o número, mais importante é o título.

Parágrafo serve para definir parágrafos e é indicado pela letra p.

É também possível alterar letras de textos. Como, por exemplo, escrever em itálico utilizando "i" ou "em"; escrever desta forma com "b"; ou até sublinhar textos com "u".
